Product Components
__________________

All the binaries delivered are standalone programs.

Delivered binary masmx contains all the features described in the
manual.

imx converts the output of masmx and masmz to 16-bit address I-Records.

mmx converts the output of masmx and masmz to 32-bit address S-Records.

The masmx feature absent from masmz is hierarchical data structure.
The directives $TREE $BRANCH $ROOT are therefore absent from masmz.

Binaries delivered for 640K DOS are the subset masmz, imx and mmx.
masmz for 640K DOS does not have the internal linker. masmz for 640K DOS
generates relocatable as well as absolute code for reasons of uniformity.

The relocatable code model and internal linker are in both masmx
and masmz for all developer platforms except 640K MS DOS.

Therefore all features of all binaries are available for Unix, Windows,
and Extended-Memory DOS.
